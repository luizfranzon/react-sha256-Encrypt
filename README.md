<!-- GETTING STARTED -->
## Getting Started

**1- Clone this repository:**
```bash
git clone https://github.com/luizfranzon/react-sha256-Encrypt.git
```
**2- Install the dependencies:**
```bash
npm install
```
**3- Run the application:**
```bash
npm run dev
```
